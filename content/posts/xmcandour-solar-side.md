### 房屋尺寸

以下尺寸必须在勘踏图纸上标注，否则延迟出图。

| 位置     | 南北(米) | 东西(米) | 备注                   |
| -------- | -------- | -------- | ---------------------- |
| 屋面     |          |          | 默认内径               |
| 女儿墙宽 |          |          | 每个方向都要在图纸标注 |
| 女儿墙高 |          |          | 不填默认低于20厘米     |
| 屋檐     |          |          | 每个方向都要在图纸标注 |
| 超出屋檐 |          |          | 不填默认无要求         |

### 南部立柱

南部立柱最低2米，如有特殊要求需注明。
{{< notice warning >}}
如果忘记注明，需增加50元设计费
{{< /notice >}}
